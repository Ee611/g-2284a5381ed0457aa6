# Change Today's Gift Photo

Use this when you want the gift page to show a new photo.

1. Open the GitHub repository on your phone.
2. Go to `p/b3a84384cd134556a3c4e661c5073caa/daily-photos/uploads`.
3. Tap `Add file`, then upload one photo from your phone.
4. Commit the upload.

GitHub will refresh the photo list automatically. After a short wait, the gift page will show the newest uploaded photo. If no photo has been uploaded, the page keeps showing the original wink GIF.
